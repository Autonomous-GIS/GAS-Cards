VectorAnalysisAgent

Version 1.3.0

This agent performs vector-based spatial analysis and transformation. It supports operations such as clipping, joining, buffering, intersecting, distance analysis, filtering, and aggregation, returning either GeoJSON for spatial outputs or CSV for tabular outputs.

RasterAgent

Version 1.2.0

This agent performs raster-focused and mixed raster-vector spatial analysis. It supports tasks such as clipping, raster calculations, terrain-style operations, rasterization, and combined raster-vector workflows, returning outputs in formats such as GeoTIFF for raster products or GeoJSON for vector products.

PasdaDiscoveryAgent

Version 1.0.0

This agent specializes in discovering and downloading Pennsylvania geospatial data from PASDA services (https://www.pasda.psu.edu/). It can inspect available services, review candidate layers, sample their attributes, and retrieve the selected layer as GeoJSON.

MappingAgent

Version 1.0.0

This agent creates visual outputs from prepared datasets, including maps and charts, returning result as a PNG image or an HTML file for interactive maps. It chooses an appropriate visualization style based on the task and available data structure. It is most useful at the final presentation stage of a workflow when analysis results need to be communicated clearly.

MapProjectionAgent

Version 1.0.0

This agent determines an appropriate coordinate reference system for a spatial task based on the user request and the dataset context. It evaluates candidate CRS options and applies the selected transformation. It is intended for reprojection tasks where CRS choice affects spatial accuracy and analysis quality.

DataRetrieverAgent

Version 1.0.0

This agent interprets a user data request, matches it to the most suitable supported source, and uses the handbook guidance for that source. It generates and executes download logic, returning the requested geospatial dataset. It is best suited for general-purpose data acquisition from supported external sources.
