VectorAnalysisAgent

Version 1.3.0

This agent performs vector-based spatial analysis and transformation using code-driven workflows. It supports operations such as clipping, joining, buffering, intersecting, distance analysis, filtering, and aggregation. It can return either GeoJSON for spatial outputs or CSV for tabular outputs derived from the analysis.

RasterAgent

Version 1.2.0

This agent performs raster-focused and mixed raster-vector spatial analysis through code-driven execution. It supports tasks such as clipping, raster calculations, terrain-style operations, rasterization, and combined raster-vector workflows. It returns outputs in formats appropriate to the result, such as GeoTIFF for raster products or GeoJSON for vector products.

PasdaDiscoveryAgent

Version 1.0.0

This agent specializes in discovering and downloading Pennsylvania geospatial data from PASDA services (https://www.pasda.psu.edu/). It can inspect available services, review candidate layers, sample their attributes, and retrieve the selected layer as GeoJSON. It is the preferred option when the requested data is likely to exist within PASDA.

MappingAgent

Version 1.0.0

This agent creates visual outputs from prepared datasets, including maps and charts, and writes the result as a PNG image. It chooses an appropriate visualization style from the task and available data structure. It is most useful at the final presentation stage of a workflow when analysis results need to be communicated clearly.

MapProjectionAgent

Version 1.0.0

This agent determines an appropriate coordinate reference system for a spatial task based on the user request and the dataset context. It evaluates candidate CRS options and applies the selected transformation to a single vector dataset. It is intended for reprojection tasks where CRS choice affects spatial accuracy and analysis quality.

DataRetrieverAgent

Version 1.0.0

This agent interprets a user data request, matches it to the most suitable supported source, and uses the handbook guidance for that source. It generates and executes download logic, then normalizes the output into a usable geospatial dataset. It is best suited for general-purpose data acquisition from supported external sources.
